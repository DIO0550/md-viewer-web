# md-viewer-web

GitHub Pages + React + Vite で構築するマークダウンビューア・エディタ。

## 概要

マークダウンファイルの表示（プレビュー）と編集ができるWebアプリケーションです。

## 技術スタック

- **フロントエンド**: React + Vite
- **ホスティング**: GitHub Pages
- **開発環境**: DevContainer (Docker)

## 開発環境のセットアップ

DevContainer を使って開発環境を構築できます。

1. VS Code で Remote - Containers 拡張機能をインストール
2. リポジトリをクローン
3. VS Code でフォルダを開き「Reopen in Container」を選択
